 NELLY VOITURE

Un projet web réalisé dans le cadre du  S4 Fullstack : Flexbox & Grid.  
Ce site présente une page unique avec plusieurs sections modernes, inspirées des layouts classiques du web.


 Contenu du projet

 1. Barre de navigation (Flexbox)
- Logo à gauche, liens à droite.
- La barre reste collée en haut quand on défile la page (sticky).
- Utilisation de Flexbox pour aligner les éléments.

 2. Hero Section
- Grande image en arrière-plan.
- Texte superposé : titre accrocheur, phrase descriptive, deux boutons.
- Mise en avant du style "landing page".

3. Holy Grail Layout (Grid)
- Découpe en 5 zones : `header`, `nav`, `main`, `aside`, `footer`.
- Utilisation de CSS Grid pour organiser la page comme un journal.

 4. Bento Gallery
- Galerie asymétrique avec 5 images (plats).
- Une image plus grande que les autres pour créer un effet moderne.
- Utilisation de **Grid** pour la mosaïque.

 5. Card Deck
- Série de 3 cartes identiques : image, titre, texte, bouton.
- Alignement des boutons en bas grâce à Flexbox.
- Adaptatif : toutes les cartes gardent la même hauteur.

 6. Footer global
- Pied de page simple en bas de la page.




 Technologies utilisées
- HTML5 : structure  des sections.
- CSS3 : Flexbox et Grid pour la mise en page.
- Unsplash : images telechargeés.



 Installation et lancement

1. Clone ou télécharge le projet.
2. Ouvre le dossier dans Visual Studio Code.
3. Installe l’extension Five Server.
4. Clique sur Go Live pour lancer le serveur local.


 Objectif pédagogique
Ce projet montre comment :
- Utiliser **Flexbox** pour aligner des éléments en ligne ou en colonne.
- Utiliser **Grid** pour organiser une page en plusieurs zones.
- Créer des layouts modernes (Hero, Holy Grail, Bento, Cards).
- Structurer une page complète sans media queries.


 Aperçu du site
- Barre de navigation collée en haut.
- Hero avec image et texte superposé.
- Holy Grail en 5 zones.
- Bento Gallery asymétrique.
- Card Deck avec 3 cartes.
- Footer global.
